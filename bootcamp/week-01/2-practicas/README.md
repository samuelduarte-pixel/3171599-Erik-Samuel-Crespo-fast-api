## 📖 Metodología

Cada ejercicio sigue el formato de **tutorial guiado**:

1. **Lee** las instrucciones en el `README.md` del ejercicio
2. **Abre** el archivo `starter/main.py`
3. **Descomenta** el código paso a paso
4. **Ejecuta** para verificar que funciona
5. **Experimenta** modificando el código

> 💡 **Tip**: No copies y pegues. Escribe el código manualmente para mejor retención.

## ✅ Criterios de Completitud

Para considerar un ejercicio completado:

- [ ] Código descomentado y funcionando

- [ ] Puedes explicar el concepto a alguien más
- [ ] Has experimentado con variaciones
