## 🚀 Instrucciones Generales

### Requisitos

- Docker y Docker Compose instalados ([Bootcamp Docker](https://github.com/ergrato-dev/bc-docker))
- Editor de código (VS Code recomendado)

### Cómo trabajar cada ejercicio

1. **Navegar** a la carpeta del ejercicio
2. **Leer** el README.md con las instrucciones
3. **Abrir** `starter/main.py`
4. **Descomentar** el código según las instrucciones
5. **Ejecutar** con Docker:
   ```bash
   docker compose up --build
   ```
6. **Probar** en http://localhost:8000/docs

### Estructura de cada ejercicio

```
ejercicio-XX/
├── README.md           # Instrucciones paso a paso
└── starter/
    ├── main.py         # Código para descomentar
    ├── pyproject.toml
    ├── Dockerfile
    └── docker-compose.yml
```
