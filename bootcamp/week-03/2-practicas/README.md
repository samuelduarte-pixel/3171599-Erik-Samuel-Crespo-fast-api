---

## 📋 Instrucciones Generales

1. Cada ejercicio tiene su propia carpeta con `starter/`
2. Lee el README.md del ejercicio
3. Descomenta el código en `starter/main.py`
4. Ejecuta con Docker para verificar

### Comandos Docker

```bash
# Desde la carpeta del ejercicio
cd ejercicio-XX-nombre/starter

# Construir y ejecutar
docker compose up --build

# Ver en navegador
# http://localhost:8000
# http://localhost:8000/docs (Swagger UI)

# Detener
docker compose down
```
